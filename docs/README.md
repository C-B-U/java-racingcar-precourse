## 기능 목록

- [x] 이름 입력 안내 출력
- [x] 이름 입력
  - [x] `,`로 이름을 나누어 입력하지 않은 경우 예외 발생
  - [x] 이름이 5글자가 넘을 경우 예외 발생
- [x] 횟수 입력 안내 출력
- [x] 횟수 입력
  - [x] 숫자로 입력하지 않은 경우 예외 발생
- [x] 입력받은 횟수만큼 자동차 이동
    - [x] 랜덤으로 숫자를 뽑는다.
    - [x] 뽑은 숫자가 4이상일 경우 이동한다.
- [x] 진행 내용 출력
  - [x] 달린 만큼 `-`를 출력한다
  - [x] `이름: --` 형식으로 출력한다.
- [x] 최종 결과
    - [x] 가장 멀리 나간 자동차가 무엇인지 구한다.
- [x] 최종 우승자 출력
  - [x] 우승자가 한 명일 경우 그냥 출력
  - [x] 우승자가 여러명일 경우 `,`로 나누어 출력
