## 기능 목록

- [x] 이름 입력 안내 출력
- [x] 이름 입력
  - [ ] `,`로 이름을 나누어 입력하지 않은 경우 예외 발생
  - [x] 이름이 5글자가 넘을 경우 예외 발생
- [x] 횟수 입력 안내 출력
- [ ] 횟수 입력
  - [ ] 숫자로 입력하지 않은 경우 예외 발생
- [ ] 입력받은 횟수만큼 자동차 이동
    - [ ] 랜덤으로 숫자를 뽑는다.
    - [ ] 뽑은 숫자가 4이상일 경우 이동한다.
- [ ] 최종 결과
    - [ ] 가장 멀리 나간 자동차가 무엇인지 구한다.
- [ ] 최종 우승자 출력
  - [ ] 우승자가 한 명일 경우 그냥 출력
  - [ ] 우승자가 여러명일 경우 `,`로 나누어 출력
